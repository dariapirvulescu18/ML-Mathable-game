2. Cum se ruleaza proiectul
In fisierul task_1_2_3.ipynb in ultima celula este scrisa logica pentru a genera fisierul de solutii:
Pe linia 13 exista variabila fisier_date aceasta trebuie inlocuita cu calea fisierului in care sunt datele de intrare
Pe linia 14 exista variabila game_numbers aceasta trebuie inlocuita cu numarul de jocuri (default este 4)

Toate taskurile (1,2 si 3) se executa prin rularea intregului fisier task_1_2_3.ipynb si genereaza un fisier de forma 351_Pirvulescu_Daria unde se afla predictiile si scorurile.

Daca proiectul se ruleaza mai mult de o singura data, inainte de al rula trebuie sters folderul
351_Pirvulescu_Daria (daca acesta s-a creat), deoarce fisierul cu scoruri este creat prin adaugare.

*Mai exista si fisierul templates.ipynb. Acesta nu trebuie rulat, ci este doar o dovada a prelucrari templaturilor atasate in folderul templates.
